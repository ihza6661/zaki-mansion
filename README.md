# Zaki Mansion Elegance

A modern and elegant landing page for the "Zaki Mansion Elegance" real estate project, designed to attract potential buyers and provide them with essential information about the property.

## Key Features

- **Hero Section:** A captivating introduction to the property with a stunning visual.
- **Unit Types:** Detailed presentation of the available housing units (Type 36, 54, 72).
- **Amenities:** A showcase of the exclusive facilities like a swimming pool, garden, and 24/7 security.
- **KPR Calculator:** An integrated mortgage calculator to help potential buyers estimate their monthly payments.
- **Location:** An embedded map showing the strategic location of the property.
- **Contact Information:** Easy access to contact details for inquiries.
- **Responsive Design:** Fully responsive layout for optimal viewing on all devices.

## Tech Stack

This project is built with a modern frontend toolchain:

- **Framework:** [React](https://react.dev/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **UI Components:** [shadcn/ui](https://ui.shadcn.com/)

## Getting Started

Follow these instructions to set up and run the project locally.

### Prerequisites

- [Node.js](https://nodejs.org/) (version 18 or higher recommended)
- [npm](https://www.npmjs.com/) or any other package manager like [pnpm](https://pnpm.io/) or [yarn](https://yarnpkg.com/)

### Installation

1.  **Clone the repository:**
    ```sh
    git clone https://github.com/ihza6661/zaki-mansion.git
    ```

2.  **Navigate to the project directory:**
    ```sh
    cd zaki-mansion
    ```

3.  **Install dependencies:**
    ```sh
    npm install
    ```

### Running the Development Server

To start the local development server with hot-reloading:

```sh
npm run dev
```

The application will be available at `http://localhost:5173` (or another port if 5173 is in use).

### Building for Production

To create a production-ready build of the application:

```sh
npm run build
```

The optimized static files will be generated in the `dist/` directory.

### Linting

To check the code for any linting issues:

```sh
npm run lint
```
